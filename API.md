# API Документация

Документация по API приложения Flask Lab Project.

## Общая информация

- **Базовый URL**: `http://localhost:5000`
- **Формат данных**: `application/json`
- **Кодировка**: `UTF-8`

## Аутентификация

На текущий момент API не требует аутентификации. В будущих версиях планируется добавление JWT-токенов.
