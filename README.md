# Flask Lab Project

Веб-приложение на Flask для демонстрации работы с репозиторием GitHub и системами контроля версий.

## Описание

Проект представляет собой минимально жизнеспособный продукт (MVP) системы управления элементами с базовым функционалом:
- Добавление новых элементов
- Просмотр списка элементов
- Удаление элементов
- Система уведомлений
- Простой API endpoint

## Технологии

- Python 3.8+
- Flask 2.3.3
- HTML5 / CSS3 / JavaScript
- Git / GitHub
