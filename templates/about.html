О проекте

Назначение

Данное приложение демонстрирует базовые принципы разработки веб-систем на Flask, организации работы с репозиторием GitHub и использования системы контроля версий Git.

Технологии

  • Python 3.8+
  • Flask 2.3.3
  • HTML5 / CSS3
  • Git / GitHub

Функциональность

  • Добавление новых элементов
  • Просмотр списка элементов
  • Удаление элементов
  • Система уведомлений
  • API endpoint для проверки статуса

Автор

Лабораторная работа выполнена в рамках учебного курса по разработке программного обеспечения.
